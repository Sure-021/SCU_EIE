# SCU_EIE 
**介绍：本项目主要展示了四川大学电子信息学院的主要课程的资料以及作业参考，希望可以给各位的学习一点帮助（虽然大部分都是期末复习），各位同学如果觉得有帮助的话可以推荐给身边的人，然后最好点个赞。
**使用办法：点击你想查看的科目的对应学期（如果有课程时间和内容更新请发邮件联系我们），如果没有找到请查看相邻学期，注意新开的课程可能会缺失对应的资料。由于咱们学院有卓越工程师班的存在，尽量在文件中标注卓工班使用，一般情况下卓工班的文件难度稍大于普通班，请各位同学酌情取用。
**更新：如果你有合适的资料文件想要上传到这个库中用于帮助新来的电子学子，请将您的文件作为附件发邮件到：zkx2004@foxmail.com，我会在本文件（README）中的最后填写您的称呼，请在邮件正文中填写您希望登上的称呼和您的文件的主要内容，例如：“射频期末试题 From Mr.sure 2022”.
